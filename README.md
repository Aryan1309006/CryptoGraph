# CryptoGraph

A React + Vite crypto dashboard app ready for deployment.

## Deploy on Vercel

1. Push this project to GitHub.
2. Import the repository in Vercel.
3. Vercel will detect Vite automatically.
4. Use the default build settings:
   - Build Command: npm run build
   - Output Directory: dist

A [vercel.json](vercel.json) file is included to ensure client-side routes such as /coin/:id work correctly.

## Local development

```bash
npm install
npm run dev
```
